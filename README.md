DS3424 – Computational Thinking for Science

Semester: Jan 2026
Instructor: Dr. Amit Apte
Credits: 4 (Lecture + Lab)

About This Repository

This repository contains my assignments, lab work, and notes for the course Computational Thinking for Science (DS3424).

The course studies computation as a scientific process through the framework:

Algorithms → Expression → Execution

We explore how scientific processes can be modeled as computational systems, expressed formally using the Scheme programming language, and executed on abstract and physical machines.

Course Themes

Idea of computation (Turing Machines, λ-calculus)

Universal machine concept

Algorithmic thinking from a machine perspective

Expressing processes using minimal language constructs (Scheme)

Abstraction layers and APIs

Execution models: hardware, OS, virtual machines, containers

Computing services (graphics, networking, mathematics, AI, etc.)
